# Sistema de Offboarding Inteligente

## Problema
Cuando una persona se va de la compañía, el proceso de inactivación
en los sistemas es manual y descoordinado. Frecuentemente el correo
queda activo más tiempo del necesario por falta de seguimiento.

## Solución
Aplicación web con IA que coordina y documenta el proceso completo
de salida de un empleado — generando checklists automáticos,
notificando responsables y registrando cada acción.

## Sistemas involucrados
- SAP S/4HANA
- Active Directory
- Correo corporativo
- Salesforce
- Ventas Móvil
- Acatex / Eliotex

## Alcance del curso (4 semanas)
- Checklist automático por área y sistemas
- Panel de seguimiento por empleado
- Gestión de excepciones con IA
- Historial y auditoría del proceso
