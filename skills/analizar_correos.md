# Skill: analizar_correos_gmail

## Descripcion
Analiza correos de Gmail recuperados via MCP
y genera un resumen inteligente con alertas
y recomendaciones de accion.

## Inputs requeridos
- correos: lista de correos con asunto, remitente, fecha, contenido
- pregunta: que quiere saber el usuario

## Pasos a seguir
1. Leer todos los correos del contexto
2. Identificar patrones importantes
3. Detectar alertas de seguridad o urgencias
4. Ordenar por relevancia segun la pregunta
5. Generar respuesta con formato consistente

## Output esperado
- Tabla de correos ordenada por fecha
- Alertas de seguridad si las hay
- Recomendaciones de accion concretas
- Formato markdown siempre igual
